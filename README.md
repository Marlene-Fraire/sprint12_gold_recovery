# **Proyecto: Predicción de Recuperación de Oro**

Este proyecto simula un escenario real en la industria minera, en el que se busca predecir la eficiencia del proceso de recuperación de oro a partir de datos operativos.

## **Objetivo**
Construir un modelo de aprendizaje automático que prediga dos objetivos:
  * Recuperación del concentrado rougher (rougher.output.recovery)
  * Recuperación del concentrado final (final.output.recovery)

## **Herramientas utilizadas**
  * Python 3.10
  * Pandas, NumPy, Seaborn, Matplotlib
  * Scikit-Learn
  * JupyterLab
  * Git y GitHub

## **Métrica de evaluación**
Se utiliza el sMAPE (error medio absoluto porcentual simétrico), que evalúa la precisión de las predicciones teniendo en cuenta tanto el valor real como el predicho.

## **Resultados esperados**
  * Optimizar los procesos industriales de recuperación de oro.
  * Detectar parámetros del proceso que no aportan valor predictivo.
  * Proveer un prototipo funcional para ser integrado en entornos industriales.
